# Jardín Infantil - Sistema de Gestión

Sistema web para la gestión de jardines infantiles desarrollado con React y TypeScript.

## Características

- Gestión de estudiantes con información personal y contactos de emergencia
- Administración de profesores y grupos
- Control de asistencia y notas
- Interfaz moderna y responsive

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Construcción

```bash
npm run build
```

## Estructura del Proyecto

```
src/
├── components/     # Componentes React
├── types/         # Definiciones TypeScript
├── data/          # Datos de ejemplo
├── main.tsx       # Punto de entrada
└── index.css      # Estilos globales
```
